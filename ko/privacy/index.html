개인정보 처리방침

1) 개인정보 처리자

개인정보 처리자는 다음과 같습니다.
Canario-Lab / Chrumek App
문의 이메일: chrumek@chrumek.app

2) 적용 범위와 목적

Chrumek은 ‘쓰지 않은 돈’과 실제로 따로 떼어 둔 금액, 그리고 그 확정 기록을 남기면서 저축 습관을 돕는 앱입니다.

저희는 사용자 계정을 운영하지 않으며, 사용자를 직접 식별할 수 있는 데이터를 수집하지 않습니다.

3) 앱에서 처리되는 데이터 (기기 내 로컬 저장)

앱에 입력한 데이터(예: 기록, 금액, 날짜, 카테고리, 목표, 설정)는 사용자의 기기에 로컬로 저장됩니다.

이 데이터는 자동으로 저희나 저희 서버로 전송되지 않습니다.

4) 시스템 접근이 필요한 기능

앱은 동작에 필요한 범위에서만 시스템 기능과 관련 데이터를 사용합니다.

  • 알림 (UNUserNotificationCenter) - 알림을 켜면 iOS가 사용자의 설정에 따라 알림을 전달합니다.
  • 언어 / 지역 / 통화 / 테마 - 앱의 표시와 형식을 맞추기 위해 사용되며, 저희에게 전송되지는 않습니다.

5) 버그 신고 / 고객지원 문의

‘버그 신고’ 기능을 사용하거나 고객지원에 연락하는 경우:

  • 앱이 메일 앱 안에 메일 초안을 만들 수 있습니다.
  • 그 내용에는 진단 정보(예: 앱 버전, iOS 버전, 언어 설정, 통화 설정)가 포함될 수 있으며, 전송 전에 사용자가 직접 확인할 수 있습니다.
  • 메일은 사용자가 직접 발송을 확정한 경우에만 전송됩니다.

전송된 메일은 문의 대응을 위해 사용자의 메일 서비스 제공자와 저희가 수신자로서 처리합니다.

6) 구매 및 구독 (Apple + RevenueCat)

앱에서 유료 기능(구독)을 제공하는 경우:

  • 결제는 Apple이 처리합니다 (App Store / In-App Purchases / StoreKit).
  • 앱은 제안 표시, 구독 상태 확인, Premium 권한 부여 등을 위해 RevenueCat을 구독 인프라로 사용합니다.

구독 처리 과정에서 다음과 같은 기술 데이터가 처리될 수 있습니다.

  • 구매 / 거래 ID 및 상품 정보
  • 구독 상태(활성 / 비활성, 갱신 여부)
  • 구독 상태를 연결하기 위한 기술 식별자(예: SDK가 생성한 익명 App User ID)

RevenueCat은 구독 관리 서비스를 제공하기 위해 이 데이터를 처리하며(처리자 역할), Apple은 결제 관련 데이터를 자체 규칙에 따라 처리합니다.

7) 분석 및 추적

앱은 사용자 추적을 위한 분석 도구나 광고 SDK를 사용하지 않으며, 데이터를 판매하지도 않습니다.

8) 웹사이트 (랜딩 페이지)

웹사이트를 방문하면 호스팅 제공자(예: GitHub Pages)가 보안, 진단, 서비스 지속성을 위해 IP 주소, 브라우저 종류, 요청 일시 같은 표준 기술 로그를 자동으로 처리할 수 있습니다. 처리 범위와 규칙은 호스팅 제공자의 정책을 따릅니다.

9) 데이터 수신자

데이터는 다음 경우에만 제공될 수 있습니다.

  • 결제 및 구독 관련 제공자: Apple 및 RevenueCat (구독 처리를 위해)
  • 법령상 권한이 있는 공공기관

10) 보관 기간

  • 앱에 입력한 데이터는 사용자가 삭제할 때까지 기기에 저장됩니다 (예: 기록 삭제 또는 앱 삭제 시점까지).
  • 이메일 문의 내용은 문의 대응과 필요한 권리 보호를 위해 필요한 기간 동안 보관됩니다.

11) 사용자 권리 (EU / EEA)

특정 상황에서 저희가 사용자의 개인정보를 처리하는 경우(예: 사용자가 이메일로 문의한 경우), 사용자는 다음 권리를 가집니다.

  • 열람, 정정, 삭제, 처리 제한 요청
  • 처리에 대한 이의 제기
  • 관할 감독기관에 대한 민원 제기

정보 제공 의무의 자세한 범위는 특히 GDPR 제13조를 따릅니다.

12) 보안

저희는 서비스 성격에 맞는 합리적인 기술적·조직적 보호조치를 적용합니다. 앱 데이터는 기기 안에 저장되므로, 기기 자체를 보호하는 것(암호 / Face ID)도 중요한 보안 요소입니다.

13) 정책 변경

이 정책은 앱 기능이나 법적 요구사항의 변경에 따라 업데이트될 수 있습니다. 최신 버전은 앱 웹사이트에 게시됩니다.
