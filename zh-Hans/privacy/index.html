隐私政策

1) 数据控制方

个人数据控制方为：
Canario-Lab / Chrumek App
联系邮箱：chrumek@chrumek.app

2) 范围和目的

Chrumek 应用通过记录“诱惑”和已确认的转账 / 存下金额，帮助追踪储蓄。

我们不运营用户账号，也不收集可直接识别您身份的数据。

3) 应用中处理的数据（本地存储在您的设备上）

您在应用中输入的数据（例如储蓄记录、金额、日期、类别、目标和设置）会本地存储在您的设备上。

这些数据不会自动发送给我们或我们的服务器。

4) 需要系统访问权限的功能

应用可能仅在运行所需范围内使用系统功能和相关数据：

  • 通知（UNUserNotificationCenter）- 如果您启用提醒，iOS 会根据您的设置发送通知。
  • 语言 / 地区 / 货币 / 主题 - 用于格式化应用内容和外观（不会向我们发送任何内容）。

5) 错误报告 / 联系支持

如果您使用“报告错误”选项或联系支持：

  • 应用可能会在您的邮件应用中创建邮件草稿，
  • 内容可能包含诊断信息（例如应用版本、iOS 版本或语言 / 货币设置）- 这些内容在发送前都会显示给您查看，
  • 邮件只有在您自行确认后才会发送。

已发送的邮件由您的邮件服务提供商以及作为收件人的我们处理，仅用于处理您的请求。

6) 购买和订阅（Apple + RevenueCat）

如果应用中提供付费功能（订阅）：

  • 付款由 Apple 处理（App Store / 应用内购买 / StoreKit），
  • 应用使用 RevenueCat 作为订阅基础设施提供方（包括展示优惠、验证订阅状态和授予 Premium 访问权限）。

在订阅处理过程中，可能会处理与购买相关的技术数据，例如：

  • 交易 / 购买标识符和产品信息，
  • 订阅状态（有效 / 无效、续订），
  • 用于关联订阅状态的技术用户标识符（例如 SDK 生成的匿名 App User ID）。

RevenueCat 为提供订阅处理服务而处理这些数据（作为处理方），Apple 则根据其自身规则处理付款相关数据。

7) 分析和跟踪

应用不使用分析工具或广告 SDK 跟踪用户，也不会出售数据。

8) 网站（落地页）

如果您访问我们的网站，托管服务提供商（例如 GitHub Pages）可能会自动在服务器日志中处理标准技术数据（包括 IP 地址、浏览器类型以及请求日期和时间），用于安全、诊断和服务连续性目的。此类处理的范围和规则遵循托管服务提供商的政策。

9) 数据接收方

数据仅可能披露给：

  • 付款和订阅服务提供方：Apple 和 RevenueCat（用于订阅处理），
  • 法律要求时的公共机关。

10) 存储期限

  • 应用中输入的数据会本地存储，直到您删除它（例如删除记录或删除应用）。
  • 电子邮件往来会在处理请求及在必要时抗辩索赔所需期间内保存。

11) 用户权利（欧盟 / 欧洲经济区）

如果在特定情况下我们处理您的个人数据（例如您向我们发送电子邮件时），您有权：

  • 访问、更正、删除或限制处理，
  • 提出反对，
  • 向有管辖权的监管机构投诉。

信息义务的范围尤其依据 GDPR 第 13 条。

12) 安全

我们采用与服务性质相适应的合理组织和技术措施。由于应用数据本地存储，保护您的设备（密码 / Face ID）也是安全的重要部分。

13) 政策变更

本政策可能会因应用功能或法律要求变化而更新。当前版本将在应用网站上发布。
