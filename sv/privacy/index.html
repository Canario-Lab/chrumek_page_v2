Integritetspolicy

1) Personuppgiftsansvarig

Personuppgiftsansvarig är:
Canario-Lab / Chrumek App
Kontakt via e-post: chrumek@chrumek.app

2) Omfattning och syfte

Chrumek-appen hjälper dig att följa sparande genom att registrera "frestelser" och bekräftade överföringar / belopp som lagts undan.

Vi driver inga användarkonton och samlar inte in data som gör att vi direkt kan identifiera dig.

3) Data som behandlas i appen (lokalt på din enhet)

Data som du anger i appen (till exempel sparposter, belopp, datum, kategorier, mål och inställningar) lagras lokalt på din enhet.

Den skickas inte automatiskt till oss eller till våra servrar.

4) Funktioner som kräver systemåtkomst

Appen kan använda systemfunktioner och relaterade data endast i den utsträckning som krävs för att den ska fungera:

  • Notiser (UNUserNotificationCenter) - om du aktiverar påminnelser levererar iOS notiser enligt dina inställningar.
  • Språk / region / valuta / tema - används för att formatera innehåll och utseende i appen (utan att något skickas till oss).

5) Felrapporter / kontakt med support

Om du använder alternativet "Rapportera ett fel" eller kontaktar supporten:

  • kan appen skapa ett utkast till e-post i din e-postapp,
  • kan innehållet omfatta diagnostisk information (till exempel appversion, iOS-version eller språk- / valutainställningar) - exakt det du kan se i meddelandet innan du skickar det,
  • skickas e-postmeddelandet endast om du själv godkänner det.

Ett skickat e-postmeddelande behandlas av din e-postleverantör och av oss som mottagare, endast för att hantera din förfrågan.

6) Köp och prenumerationer (Apple + RevenueCat)

Om betalda funktioner (prenumerationer) finns i appen:

  • hanteras betalningar av Apple (App Store / köp i appen / StoreKit),
  • använder appen RevenueCat som infrastrukturleverantör för prenumerationer (inklusive visning av erbjudanden, verifiering av prenumerationsstatus och tilldelning av Premium-åtkomst).

Som en del av prenumerationshanteringen kan tekniska köpdata behandlas, till exempel:

  • transaktions- / köpidentifierare och produktinformation,
  • prenumerationsstatus (aktiv / inaktiv, förnyelser),
  • en teknisk användaridentifierare som används för att koppla prenumerationsstatus (till exempel ett anonymt App User ID som genereras av SDK:n).

RevenueCat behandlar dessa data för att tillhandahålla prenumerationshantering (som personuppgiftsbiträde), och Apple behandlar betalningsrelaterade data enligt sina egna regler.

7) Analys och spårning

Appen använder inte analysverktyg eller reklam-SDK:er för att spåra användare och säljer inte data.

8) Webbplats (landningssida)

Om du besöker vår webbplats kan värdleverantören (till exempel GitHub Pages) automatiskt behandla standardmässiga tekniska data i serverloggar (inklusive IP-adress, webbläsartyp samt datum och tid för begäran) för säkerhet, diagnostik och kontinuitet i tjänsten. Omfattningen och reglerna för sådan behandling följer värdleverantörens policyer.

9) Mottagare av data

Data kan endast lämnas ut till:

  • betalnings- och prenumerationsleverantörer: Apple och RevenueCat (för prenumerationshantering),
  • offentliga myndigheter, om lagen kräver det.

10) Lagringstid

  • Data som anges i appen lagras lokalt tills du tar bort den (till exempel genom att radera poster eller radera appen).
  • E-postkorrespondens lagras så länge det behövs för att hantera förfrågan och försvara oss mot anspråk (om det behövs).

11) Användarens rättigheter (EU / EES)

Om vi i ett visst fall behandlar dina personuppgifter (till exempel när du skickar e-post till oss) har du rätt att:

  • få tillgång till, rätta, radera eller begränsa behandlingen,
  • invända mot behandlingen,
  • lämna in klagomål till behörig tillsynsmyndighet.

Omfattningen av informationsskyldigheterna följer särskilt av artikel 13 i GDPR.

12) Säkerhet

Vi använder rimliga organisatoriska och tekniska åtgärder som är anpassade till tjänstens natur. Eftersom appdata lagras lokalt är skyddet av din enhet (lösenkod / Face ID) också en viktig del av säkerheten.

13) Ändringar i policyn

Denna policy kan uppdateras när appens funktioner eller rättsliga krav ändras. Den aktuella versionen publiceras på appens webbplats.
