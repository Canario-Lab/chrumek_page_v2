「Chrumek」アプリ利用規約

発効日: 2026年4月23日
バージョン: 1.1

1. 提供者

  1. 「Chrumek」アプリ（以下「本アプリ」）の提供者は以下のとおりです。
    Canario-Lab / Chrumek App
    メール: chrumek@chrumek.app
    （以下「提供者」）
  2. 本規約は、本アプリの無料機能および有料機能の利用条件を定めるものです。

2. 定義

本規約において、以下の語は次の意味で用います。

  1. ユーザー
    Apple 端末上で本アプリを利用する者。
  2. 記録
    購入を見送ったこと、より安い選択肢を選んだこと、サブスクリプションを解約したことなど、ユーザーが本アプリに入力する節約関連の情報。
  3. 「貯金に移しました」
    記録した金額を実際に取り分けたことをアプリ上で確定する操作。未確定の一覧から履歴へ移り、実際に貯金した金額として扱われます。
  4. Premium
    App Store で利用可能なオファーに応じて、サブスクリプションまたは買い切りのライフタイムアクセス購入により利用できる、本アプリの全機能への有料アクセス。

3. 技術要件

  1. 本アプリの利用には、対応する Apple 端末と、本アプリが対応する iOS バージョンが必要です。
  2. 一部の機能（特に購入、購入の復元、または Premium 状態の確認）には、インターネット接続と Apple ID が必要です。

4. アプリの性質

  1. 本アプリは、使わずに済んだお金を記録し、実際に貯金へ移したことを確認することで、貯金習慣をサポートするものです。
  2. 本アプリは金融サービス、銀行、投資助言サービスではありません。実際の送金を代行したり、取引の仲介をしたりするものでもありません。
  3. 提供者は、ユーザーが特定の貯金成果を達成することを保証しません。

5. データとプライバシー

  1. アプリはアカウント登録を必要としません。
  2. アプリに入力されたデータは、原則として利用者の端末内にローカル保存されます。
  3. アプリは、プライバシーポリシーに従い、機能の利用に関する匿名の分析イベントを収集する場合があります。
  4. データ処理の詳細なルールは、プライバシーポリシーに記載されています。

6. 無料機能と回数制限

  1. 本アプリには無料機能と Premium 機能があります。
  2. Premium を有効にしていないユーザーは、実際の貯金として確定できる回数が最大5回まで無料です。
  3. 5回の無料回数を使い切った後、さらに確定するには有効な Premium が必要です。
  4. 無料回数のカウンターは:
  • 実際に確定したときのみ増えます。
  • Premium アクセスを失った後も減らず、リセットもされません。

7. Premium、購入および支払い

  1. Premium は、Apple（App Store）により処理される自動更新サブスクリプション、または買い切りのライフタイムアクセス購入（non-consumable In-App Purchase）として提供される場合があります。
  2. 価格、サブスクリプションの請求期間、利用可能な購入オプションは、アプリ内の購入画面に表示され、App Store の設定に従います。
  3. 購入確定後、料金はユーザーの Apple ID に請求されます。
  4. 現在の請求期間が終了する前に Apple ID 設定で自動更新をオフにしない限り、サブスクリプションは自動更新されます。ライフタイムアクセス購入は買い切りであり、自動更新されません。
  5. 提供者は、カード情報などの支払いデータを処理しません。支払い処理は Apple が行います。

8. RevenueCat（購入と Premium アクセスの処理）

  1. 本アプリは、Premium 状態の確認、オファー表示、有効なサブスクリプションまたはライフタイムアクセス購入の認識など、購入と Premium アクセスの技術的処理のために RevenueCat を利用する場合があります。
  2. 購入処理は Apple が行い、RevenueCat はアプリ内でのアクセス状態の確認と管理を補助します。
  3. この範囲におけるデータ処理の詳細は、プライバシーポリシーに記載します。

9. 購入の復元

  1. ユーザーは、Apple の仕組みおよび本アプリの「購入を復元」機能（該当バージョンで提供される場合）を使って、過去に購入したサブスクリプションまたは買い切りのライフタイムアクセス購入を復元できます。

10. サポートと不具合報告

  1. 技術的な問題は、chrumek@chrumek.app まで報告できます。
  2. 「バグを報告」機能がある場合、本アプリは診断情報を含むメール下書きを作成することがあります。送信はユーザー自身の確認後にのみ行われます。
