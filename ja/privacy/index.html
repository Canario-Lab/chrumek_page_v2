プライバシーポリシー

1) データ管理者

個人データの管理者は以下のとおりです。
Canario-Lab / Chrumek App
連絡先メール: chrumek@chrumek.app

2) 対象範囲と目的

Chrumekは、「使わずに済んだ支出」や、実際に取り分けた金額とその確定を記録することで、貯金の習慣づくりをサポートするアプリです。

当社はユーザーアカウントを運営しておらず、利用者を直接特定できるようなデータも収集していません。

3) アプリ内で処理されるデータ（端末内のローカル保存）

アプリに入力されたデータ（たとえば記録、金額、日付、カテゴリ、目標、設定など）は、利用者の端末内にローカル保存されます。

これらのデータが自動的に当社または当社サーバーへ送信されることはありません。

4) システムアクセスを必要とする機能

アプリは、動作に必要な範囲でのみシステム機能と関連データを利用します。

  • 通知（UNUserNotificationCenter） - リマインダーを有効にした場合、iOSが利用者の設定に従って通知を配信します。
  • 言語 / 地域 / 通貨 / テーマ - アプリの表示やフォーマットを調整するために使われ、当社へ送信されることはありません。

5) バグ報告 / サポート連絡

「バグを報告」機能を使う場合、またはサポートへ連絡する場合:

  • アプリがメールアプリ内に下書きを作成することがあります。
  • その内容には診断情報（たとえばアプリのバージョン、iOSのバージョン、言語設定や通貨設定など）が含まれる場合がありますが、送信前に利用者自身が確認できます。
  • メールは利用者本人が送信を確定した場合にのみ送られます。

送信されたメールは、問い合わせ対応のために、利用者のメール提供事業者および当社が受信者として処理します。

6) 購入とサブスクリプション（Apple + RevenueCat）

アプリで有料機能（サブスクリプション）を提供する場合:

  • 決済はAppleが処理します（App Store / In-App Purchases / StoreKit）。
  • アプリは、オファー表示、サブスクリプション状態の確認、Premium権限の付与などのために、サブスクリプション基盤としてRevenueCatを利用します。

サブスクリプション処理の一環として、次のような技術データが処理される場合があります。

  • 購入・取引ID、および商品情報
  • サブスクリプションの状態（有効 / 無効、更新状況）
  • サブスクリプション状態を紐づけるための技術的な利用者識別子（たとえばSDKが生成する匿名のApp User ID）

RevenueCatはこれらのデータをサブスクリプション管理サービス提供のために処理し（処理者として）、Appleは決済関連データを独自のルールに従って処理します。

7) 解析とトラッキング

本アプリは、利用者の追跡を目的とした解析ツールや広告SDKを使用せず、データを販売することもありません。

8) ウェブサイト（ランディングページ）

当社ウェブサイトを訪れた場合、ホスティング事業者（たとえばGitHub Pages）が、セキュリティ、診断、サービス継続のために、IPアドレス、ブラウザ種別、リクエスト日時などの標準的な技術ログを自動処理することがあります。処理の範囲とルールはホスティング事業者のポリシーに従います。

9) データの受領者

データが開示される可能性があるのは、次の場合に限られます。

  • 決済およびサブスクリプション関連事業者: AppleおよびRevenueCat（サブスクリプション処理のため）
  • 法令に基づき権限を有する公的機関

10) 保存期間

  • アプリに入力したデータは、利用者が削除するまで端末内に保存されます（たとえば記録の削除やアプリの削除まで）。
  • メールのやり取りは、問い合わせ対応および必要に応じた権利保護のために必要な期間保存されます。

11) 利用者の権利（EU / EEA）

特定の状況で当社が利用者の個人データを処理する場合（たとえばメールで問い合わせを受けた場合）、利用者には次の権利があります。

  • 開示請求、訂正、削除、処理制限の請求
  • 異議申立て
  • 所管の監督機関への苦情申立て

情報提供義務の詳細は、とくにGDPR第13条に基づきます。

12) セキュリティ

当社は、サービスの性質に見合った合理的な技術的・組織的対策を講じています。アプリのデータは端末内に保存されるため、端末自体の保護（パスコード / Face ID）もセキュリティの重要な要素です。

13) ポリシーの変更

本ポリシーは、アプリ機能や法的要件の変更に応じて更新される場合があります。最新版はアプリのウェブサイトで公開します。
