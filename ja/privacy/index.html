プライバシーポリシー

1) データ管理者

本アプリのデータ管理者は以下のとおりです。
Canario-Lab / Chrumek App
連絡先メール: chrumek@chrumek.app

2) 対象と目的

Chrumek は、買わずに済んだ支出や、実際に貯金へ移した金額を記録することで、貯金習慣づくりを助けるアプリです。

本アプリではユーザー登録は不要で、原則として利用者を直接特定できる個人データは収集しません。

3) アプリ内で扱うデータ（端末内にローカル保存）

アプリに入力したデータ（記録、金額、日付、カテゴリ、目標、各種設定など）は、端末内にローカル保存されます。

これらのデータが自動的に当社や当社サーバーへ送信されることはありません。

4) システム権限を使う機能

アプリは、動作に必要な範囲でのみシステム機能や関連データを利用します。

  • 通知（UNUserNotificationCenter）
    リマインダーを有効にすると、iOS が設定に従って通知を配信します。
  • 言語 / 地域 / 通貨 / テーマ
    アプリ内の表示や書式を整えるためにのみ使われ、当社には送信されません。

5) バグ報告 / サポート連絡

「バグを報告」機能を使うか、サポートへ連絡する場合:

  • アプリはメールアプリ内で下書きを作成することがあります。
  • 下書きには診断情報（アプリのバージョン、iOS バージョン、言語 / 通貨設定など）が含まれることがあります。
  • メールは、利用者が自分で送信を確定した場合にのみ送られます。
    送信されたメールは、問い合わせ対応のために、利用者のメールサービス提供者および当社によって処理されます。

6) 購入と Premium アクセス（Apple + RevenueCat）

アプリで有料機能（サブスクリプションまたは買い切りのライフタイムアクセス購入）を提供する場合:

  • 決済は Apple（App Store / In-App Purchases / StoreKit）が処理します。
  • アプリは、オファー表示、アクセス状態の確認、有効なサブスクリプションまたはライフタイムアクセス購入の認識、Premium 権限の付与など、購入と Premium アクセス処理の技術基盤として RevenueCat を利用します。
    購入および Premium アクセスの処理において、以下のような技術データが扱われる場合があります。
  • 購入 / 取引 ID と商品情報
  • Premium アクセスの状態（有効 / 無効）、サブスクリプションの状態（更新状況など）、または買い切り購入に関する情報
  • Premium アクセス状態を紐づけるための技術的識別子（例: SDK が生成する匿名の App User ID）
    RevenueCat は購入および Premium アクセス処理機能を提供するためにこれらのデータを扱い、Apple は自社ポリシーに基づいて決済関連データを処理します。

7) 分析とトラッキング

アプリは Firebase Analytics を使用して、アプリの利用状況に関する匿名データを収集する場合があります（例：記録の追加、振替の実行、無料振替上限への到達、Premium オファー画面の表示、その画面のメインボタンのタップ）。

これらのデータには、記録の内容、金額、目標名、カテゴリ、価格、購入プラン、支払い情報、または利用者を直接識別する情報は含まれません。データは利用者の本人性や特定の端末とは関連付けられません。当社はこれらを、アプリの動作分析、安定性の改善、機能改善のためにのみ使用します。

分析データをデータ販売や広告目的のプロファイリングに使用することはありません。

8) ウェブサイト

当社ウェブサイトにアクセスした場合、ホスティング事業者（例: GitHub Pages）が、セキュリティ・診断・サービス維持の目的で、IP アドレス、ブラウザ種別、アクセス日時などの標準的な技術ログを自動的に処理することがあります。処理の範囲やルールは、ホスティング事業者のポリシーによります。

9) データの受領者

データは、以下の場合にのみ開示されることがあります。

  • 支払いおよび Premium アクセス管理サービスの提供者：Apple および RevenueCat（購入処理とアクセス状態の管理）、
  • 分析サービスの提供者：Google Firebase Analytics（アプリイベントの匿名分析）、
  • 法令により求められる場合の公的機関。

10) 保存期間

  • アプリに入力したデータは、利用者が削除するまで端末内に保存されます。
  • メールのやり取りに含まれるデータは、問い合わせ対応および必要な権利保護のために必要な期間保存されます。

11) 利用者の権利（EU / EEA）

具体的な状況で当社が利用者の個人データを処理する場合（例: メールで問い合わせを受けた場合）、利用者には次の権利があります。

  • 開示請求、訂正、削除、処理制限
  • 処理への異議申立て
  • 管轄監督機関への苦情申立て
    情報提供義務の範囲は、とくに GDPR 第13条に基づきます。

12) セキュリティ

当社は、サービスの性質に応じた合理的な技術的・組織的安全対策を講じます。アプリのデータは端末内に保存されるため、端末自体を保護すること（パスコード / Face ID など）も重要です。

13) ポリシーの変更

本ポリシーは、アプリ機能や法的要件の変更に応じて更新される場合があります。最新版はアプリのウェブサイトで公開します。
